POLÍTICA DE PRIVACIDADE

Esta política descreve como o Dash Run trata as informações do usuário.

1. DADOS COLETADOS

O Dash Run NÃO coleta dados pessoais. Não solicitamos nome, e-mail, localização ou qualquer informação de identificação pessoal.

2. ARMAZENAMENTO LOCAL

O jogo salva localmente no seu dispositivo apenas: pontuação máxima, número de partidas, conquistas desbloqueadas e estatísticas de jogo. Esses dados ficam somente no seu aparelho.

3. DADOS DE TERCEIROS

O aplicativo não integra redes sociais, serviços de análise ou plataformas de publicidade que coletam dados pessoais.

4. ANÚNCIOS

Caso anúncios sejam exibidos em versões futuras, serão não personalizados e não exigirão dados do usuário.

5. SEGURANÇA

Como nenhum dado pessoal é transmitido, o risco de exposição de informações é inexistente nesta versão do aplicativo.

6. MENORES DE IDADE

O jogo é adequado para todas as idades. Não coletamos nenhum dado de menores de idade.

7. CONTATO

Para dúvidas sobre privacidade, entre em contato pelo e-mail de suporte disponível na Play Store.
